# 专注模式使用指南

## 功能概述

专注模式允许你临时禁用所有健康提醒，专注工作一段时间。在专注模式期间，久坐提醒和喝水提醒都会被暂停。

## 主要特性

- 🎯 **临时禁用提醒**：在专注模式期间，所有健康提醒都会被暂停
- ⏱️ **自定义时长**：可以设置1-480分钟的专注时长
- 📊 **状态栏显示**：插件启动后自动在状态栏显示专注模式按钮，实时显示状态和剩余时间
- 🔄 **自动恢复**：专注时间结束后，自动恢复所有提醒

## 使用方法

### 启动专注模式

有两种方式启动专注模式：

1. **通过命令面板**：
   - 按 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (Mac)
   - 输入 "Moved Yet: Toggle Focus Mode"
   - 输入专注时长（分钟）

2. **通过状态栏**：
   - 插件启动后，状态栏右侧会自动显示 "🎯 进入专注模式" 按钮
   - 点击该按钮
   - 输入专注时长（分钟）

### 退出专注模式

有两种方式退出专注模式：

1. **手动退出**：
   - 点击状态栏的 "🎯 专注模式 (剩余时间)" 按钮
   - 或通过命令面板执行 "Moved Yet: Toggle Focus Mode"

2. **自动退出**：
   - 等待专注时间结束，系统会自动退出并恢复提醒

## 配置选项

在 VS Code 设置中可以配置：

- **专注模式默认时长** (`movedYet.focusModeDefaultDuration`)
  - 默认值：60 分钟
  - 范围：1-480 分钟
  - 说明：启动专注模式时的默认时长

## 状态栏显示

- **插件启动时**：自动显示 "🎯 进入专注模式" 按钮
- **未启动时**：显示 "🎯 进入专注模式"
- **运行中**：显示 "🎯 专注模式 (剩余X分钟)"
- **提示信息**：鼠标悬停可查看详细信息

## 使用场景

- 📝 **重要会议**：会议期间不被提醒打扰
- 💻 **深度编码**：需要长时间专注写代码
- 🎯 **紧急任务**：处理紧急问题时暂停提醒
- 📚 **学习时间**：专注学习新技术

## 注意事项

1. 专注模式期间，工作时长仍会正常统计
2. 专注模式不会影响历史记录功能
3. 专注模式结束后，提醒计时器会自动重置
4. 可以随时手动退出专注模式

## 常见问题

**Q: 专注模式会影响工作时长统计吗？**
A: 不会。专注模式只是暂停提醒，工作时长仍会正常统计。

**Q: 可以设置多长时间的专注模式？**
A: 可以设置1-480分钟（最长8小时）的专注时长。

**Q: 专注模式期间可以手动退出吗？**
A: 可以。点击状态栏按钮或通过命令面板即可退出。

**Q: 专注模式结束后需要手动恢复提醒吗？**
A: 不需要。专注模式结束后会自动恢复所有提醒。

## 技术实现

- 专注模式通过暂停计时器实现
- 状态栏每分钟更新一次剩余时间
- 使用 setTimeout 实现自动结束
- 与其他功能（活动检测、历史记录等）完全兼容
