# Change Log

### v0.0.1 (2026-01-30)

#### 项目信息
-   🔄 项目名称：Moved Yet
-   🇨🇳 中文名称：动了么？
-   📝 提供久坐和喝水的强制提醒功能

#### 新增功能

-   ✨ 添加久坐提醒和喝水提醒功能，分别默认为 60 分钟和 45 分钟
-   🖼️ 添加全屏模态窗口，确保提醒不会被忽略
-   🔒 添加 UI 锁定功能，必须确认提醒才能继续工作
-   ⏱️ 添加 3 秒倒计时确认机制，防止误操作
-   🎨 添加美观的 UI 设计，包括渐变背景和毛玻璃效果
-   🌐 添加中英文双语言支持（zh-CN 和 en）
-   📋 添加命令支持：重置计时器和显示状态
-   📊 添加状态栏项目，显示当前计时器状态
-   ⚙️ 添加可配置选项，允许自定义提醒间隔和开关控制
-   📝 添加详细的 README.md 文档
-   📈 添加提醒历史记录功能
-   🔔 添加渐进式提醒机制（状态栏→通知→全屏）
-   📊 添加运动数据可视化仪表盘

#### 文档更新

-   📦 添加完整的编译 .vsix 包教程
-   🛠️ 更新开发指南，包含详细的编译步骤
-   📚 同步所有相关文档，确保信息一致性
-   🔧 添加常见问题解决方案和故障排除指南
-   🎯 提供多种编译方法：vsce、手动脚本、Docker
-   📋 添加自动化编译配置（GitHub Actions）

#### 技术实现

-   使用 VS Code WebView API 实现全屏提醒
-   实现独立的计时器系统，确保提醒互不干扰
-   添加配置监听，实时响应用户设置变更
-   使用 Chart.js 实现数据可视化
-   实现数据持久化存储
-   📦 提供 `package-vsix.sh` 手动打包脚本
-   🔧 配置完整的 TypeScript 编译环境
