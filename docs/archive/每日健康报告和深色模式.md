# 每日健康报告和深色模式功能说明

## 📊 每日健康报告

### 功能概述
每日健康报告会在**每天9点后首次启动 VS Code 时**自动显示，为你提供**昨天**的健康数据总结和改进建议。

### 显示规则
- **时间条件**: 当前时间必须在9点或之后
- **频率**: 每天只显示一次（9点后首次启动）
- **数据来源**: 显示昨天的健康数据
- **数据要求**: 昨天必须有健康活动记录（起身或喝水）
- **延迟显示**: 启动后延迟5秒显示，避免打扰

### 主要特性

#### 1. 健康评分系统
- **评分范围**: 0-100分
- **评分维度**:
  - 起身次数（40分）：理想为每小时起身1次
  - 喝水次数（40分）：理想为每小时喝水1-2次
  - 工作时长（20分）：理想为4-8小时

#### 2. 健康评级
- 🌟 **优秀** (90-100分): 健康习惯非常好
- 😊 **良好** (75-89分): 健康习惯良好
- 😐 **一般** (60-74分): 有改进空间
- 😟 **需改进** (0-59分): 需要更多关注健康

#### 3. 智能建议
根据你的数据自动生成个性化建议：
- 起身次数不足时，建议增加活动频率
- 喝水次数不足时，提醒定时补充水分
- 工作时长过长时，建议增加休息
- 表现良好时，给予鼓励

#### 4. 数据展示
- 📊 圆形进度条显示健康评分
- 🪑 起身次数统计
- 💧 喝水次数统计
- ⏱️ 工作时长统计

### 使用方法

#### 自动显示
- **时间**: 每天9点后首次启动 VS Code
- **内容**: 显示昨天的健康数据
- **条件**: 昨天有健康活动记录
- **延迟**: 启动后5秒自动显示
- **频率**: 每天只显示一次

#### 手动查看
使用命令面板（`Ctrl+Shift+P` / `Cmd+Shift+P`）：
```
Moved Yet: View Daily Health Report
或
动了么？: 查看每日健康报告
```

**注意**: 手动查看会显示昨天的数据，如果昨天没有数据则不会显示。

#### 配置选项
```json
{
    "movedYet.enableDailyReport": true  // 启用/禁用每日报告
}
```

### 报告内容示例


- ✅ 按钮样式适配

### 使用的 VS Code 主题变量

```css

--vscode-editor-inactiveSelectionBackground

/* 前景色 */
--vscode-editor-foreground
--vscode-descriptionForeground

/* 按钮 */
```

## 🔧 配置示例

### 完整配置
```json
{
    // 基础提醒
    "movedYet.sitReminderInterval": 60,
    "movedYet.drinkReminderInterval": 45,
    "movedYet.enableSitReminder": true,
    "movedYet.enableDrinkReminder": true,
    
    // 渐进式提醒
    "movedYet.enableProgressiveReminder": true,
    "movedYet.progressiveReminderLevel1Duration": 5,
    "movedYet.progressiveReminderLevel2Duration": 5,
    
    // 活动检测
    "movedYet.enableActivityDetection": true,
    "movedYet.inactivityResetTime": 5,
    
    // 历史记录
    "movedYet.enableHistory": true,
    
    // 每日报告（新功能）
    "movedYet.enableDailyReport": true,
    
    // 语言
    "movedYet.language": "zh-CN"
}
```

### 禁用每日报告
如果你不想看到每日报告，可以禁用：
```json
{
    "movedYet.enableDailyReport": false
}
```
