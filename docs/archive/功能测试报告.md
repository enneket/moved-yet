## 🚀 安装和使用指南

### 安装方法
1. **从VSIX安装**: `code --install-extension moved-yet-0.0.1.vsix`
2. **VS Code界面**: 扩展面板 → "..." → "从VSIX安装"

### 快速开始
1. 安装后插件自动激活
2. 默认60分钟久坐提醒，45分钟喝水提醒
3. 使用 `Ctrl+Shift+P` 搜索 "动了么" 查看所有功能

### 自定义配置
在VS Code设置中搜索 "movedYet" 可配置：
- 提醒间隔时间
- 启用/禁用特定提醒
- 界面语言
- 渐进式提醒设置
