ci: Add continue-on-error for marketplace publishing steps

允许 VS Code Marketplace 和 Open VSX Registry 发布步骤失败而不影响整个发布流程。

问题：
- GitHub Actions 在发布到 VS Code Marketplace 时因为 PAT 验证失败而导致整个工作流失败
- 即使 GitHub Release 创建成功，但工作流显示为失败状态

解决方案：
- 在 Publish to VS Code Marketplace 步骤添加 continue-on-error: true
- 在 Publish to Open VSX Registry 步骤添加 continue-on-error: true

效果：
- 即使没有配置 VSCE_PAT 或 OVSX_PAT，GitHub Release 仍然可以成功创建
- 市场发布步骤失败不会影响整个工作流的成功状态
- 用户可以直接从 GitHub Release 页面下载 VSIX 文件

Changes:
- .github/workflows/release.yml: 添加 continue-on-error 标志
