## 🏗️ 技术架构

### 📁 项目结构
```
src/
├── extension.ts              # 主入口
├── models/types.ts          # 类型定义
├── services/                # 核心服务
│   ├── configService.ts     # 配置管理
│   ├── timerService.ts      # 计时器服务
│   ├── historyService.ts    # 历史记录
│   ├── statusService.ts     # 状态显示
│   ├── activityDetectionService.ts  # 活动检测
│   └── progressiveReminderService.ts # 渐进式提醒
├── ui/                      # 用户界面
│   ├── reminderUI.ts        # 提醒界面
│   └── dashboardUI.ts       # 仪表盘
└── utils/                   # 工具函数
    └── languages.ts         # 多语言支持
```

- **框架**: VS Code Extension API
- **构建**: npm + tsc
- **测试**: Mocha + VS Code Test Runner
- **代码检查**: ESLint
- **包管理**: npm/pnpm
- **CI/CD**: GitHub Actions
