# CHANGELOG 更新总结

## 📋 更新内容

### 版本号调整
- ✅ 将主版本从 `v0.0.2` 改为 `v0.0.1`
- ✅ 移除了原来的 `v0.0.1` 条目（避免重复）

### 内容整合
将原来分散在两个版本中的功能整合到单一的 `v0.0.1` 版本中：

#### 🆕 新增功能（完整列表）
- 🖱️ **智能活动检测** - 监听鼠标和键盘活动
- 🧠 **智能工作时长统计** - 基于用户活动状态的精确计时
- ⚙️ **新增配置选项** - 活动检测相关配置
- 🔍 **活动检测状态监控** - 实时状态显示
- 🎮 **工作计时控制命令** - 手动暂停/恢复功能
- 🔄 **智能重置逻辑** - 自动重置计时器

#### 🛠️ 技术改进
- 📦 升级 Node.js 到 v25.5.0
- 📦 升级 npm 到 v11.8.0
- 🔧 更新 `@types/node` 到 25.x
- 🆕 新增 `ActivityDetectionService` 单例服务

#### 🐛 修复
- ✅ **修复久坐提醒初始化顺序问题**（重要修复）
- 🌐 **统一命令标题为英文**（"Moved Yet"）
- 🔧 优化服务初始化和清理逻辑

#### 🎯 核心功能
- ✨ 久坐提醒和喝水提醒功能
- 🖼️ 全屏模态窗口
- 🌐 中英文双语言支持
- 📋 完整的命令支持
- 📊 运动数据可视化仪表盘

### 命令名称更新
所有命令标题已从中文更新为英文：
- `动了么？: 测试活动检测` → `Moved Yet: Test Activity Detection`
- `动了么？: 暂停工作计时` → `Moved Yet: Pause Work Timer`
- `动了么？: 恢复工作计时` → `Moved Yet: Resume Work Timer`
