## 🔧 配置对比

### v0.0.1 配置选项
```json
{
    "movedYet.sitReminderInterval": 60,
    "movedYet.drinkReminderInterval": 45,
    "movedYet.enableSitReminder": true,
    "movedYet.enableDrinkReminder": true,
    "movedYet.language": "zh-CN",
    "movedYet.enableProgressiveReminder": false,
    "movedYet.progressiveReminderLevel1Duration": 5,
    "movedYet.progressiveReminderLevel2Duration": 5,
    "movedYet.enableHistory": true
}
```

### v0.0.2 新增配置
```json
{
    // 继承所有 v0.0.1 配置，新增：
    "movedYet.enableActivityDetection": true,    // 启用活动检测
    "movedYet.inactivityResetTime": 5           // 无活动重置时间
}
```

## 🔄 升级建议

### 从 v0.0.1 升级到 v0.0.2

**推荐升级理由**:
1. ✅ **智能体验**: 活动检测大幅提升用户体验
2. ✅ **技术先进**: 最新 Node.js 版本，更好性能
3. ✅ **向后兼容**: 现有配置无需修改
4. ✅ **功能增强**: 新增监控和测试功能

**升级步骤**:
1. 卸载旧版本: `code --uninstall-extension Immerse.moved-yet`
2. 安装新版本: `code --install-extension moved-yet-0.0.2.vsix`
3. 重新加载 VS Code
4. 可选：启用新功能配置

**注意事项**:
- 历史数据会保留
- 配置会自动迁移
- 新功能默认启用
