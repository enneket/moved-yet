## 📊 演示 1: 每日健康报告

### 场景：查看昨天的健康数据

#### 步骤 1: 生成测试数据
```
1. 等待 1 分钟，触发久坐提醒
2. 确认提醒
3. 等待 1 分钟，触发喝水提醒
4. 确认提醒
5. 重复 2-3 次
```

#### 步骤 2: 查看报告
```
方式一：自动显示
- 重启 VS Code
- 等待 5 秒
- 报告自动弹出

方式二：手动查看
- Ctrl+Shift+P
- 输入 "daily report"
- 选择 "Moved Yet: View Daily Health Report"
```

#### 步骤 3: 讲解报告内容
```
1. 健康评分
   - 指向评分圆环
   - 说明评分范围 0-100
   - 解释当前评分

2. 评级系统
   - 指向评级图标和文字
   - 说明四个等级
   - 解释评级标准

3. 统计数据
   - 指向三个统计卡片
   - 说明起身次数
   - 说明喝水次数
   - 说明工作时长

4. 健康建议
   - 指向建议区域
   - 说明建议是如何生成的
   - 强调个性化特点

5. 操作按钮
   - 点击"查看仪表盘"
   - 展示跳转功能
```

#### 演示要点
- ✨ 强调评分的科学性
- 💡 突出建议的个性化
- 📊 展示数据的完整性
- 🎯 说明如何改进评分

## 🌙 演示 2: 深色模式适配

### 场景：展示主题自适应

#### 步骤 1: 浅色主题演示
```
1. 切换到浅色主题
   - 文件 → 首选项 → 主题
   - 选择 "Light+" 或其他浅色主题

2. 打开健康仪表盘
   - Ctrl+Shift+P
   - "Moved Yet: View Health Dashboard"

3. 讲解浅色模式效果
   - 指向背景色（浅色）
   - 指向文字颜色（深色）
   - 指向卡片样式
   - 指向按钮样式
```

#### 步骤 2: 深色主题演示
```
1. 切换到深色主题
   - 文件 → 首选项 → 主题
   - 选择 "Dark+" 或其他深色主题

2. 刷新仪表盘
   - 点击"刷新"按钮

3. 讲解深色模式效果
   - 指向背景色（深色）
   - 指向文字颜色（浅色）
   - 指向卡片样式
   - 指向按钮样式
   - 强调视觉舒适度
```

#### 步骤 3: 提醒窗口演示
```
1. 等待提醒触发
   - 或使用测试命令

2. 浅色主题下的提醒
   - 展示提醒窗口
   - 指出颜色适配

3. 深色主题下的提醒
   - 切换主题
   - 再次触发提醒
   - 对比效果
```

#### 步骤 4: 每日报告演示
```
1. 浅色主题下查看报告
   - 展示整体效果
   - 指出评分圆环颜色

2. 深色主题下查看报告
   - 切换主题
   - 再次查看报告
   - 对比效果
```

#### 演示要点
- 🌙 强调自动适配特性
- 👁️ 突出护眼效果
- 🎨 展示统一的视觉风格
- ⚡ 说明无需配置

## 🎯 演示 3: 完整工作流

### 场景：一天的健康管理

#### 早上：开始工作
```
1. 启动 VS Code
   - 等待 5 秒
   - 查看昨天的健康报告
   - 根据建议调整今天的计划

2. 设置提醒间隔
   - 根据今天的工作安排
   - 调整久坐和喝水提醒间隔
```

#### 工作中：接收提醒
```
1. 渐进式提醒演示
   - 第一级：状态栏提示
   - 第二级：通知消息
   - 第三级：全屏提醒

2. 活动检测演示
   - 离开电脑 5 分钟
   - 回来后自动重置计时器
   - 避免过期提醒
```

#### 下午：查看进度
```
1. 查看当前状态
   - Ctrl+Shift+P
   - "Moved Yet: Show Reminder Status"
   - 查看计时器状态

2. 查看今日统计
   - Ctrl+Shift+P
   - "Moved Yet: View Reminder History"
   - 查看今日数据
```

#### 晚上：数据回顾
```
1. 打开健康仪表盘
   - 查看今日统计
   - 查看本周统计
   - 查看趋势图表

2. 分析健康习惯
   - 找出规律
   - 发现问题
   - 制定改进计划
```

#### 第二天：查看报告
```
1. 启动 VS Code
   - 自动显示昨天的报告
   - 查看健康评分
   - 阅读健康建议

2. 持续改进
   - 根据评分调整习惯
   - 追踪评分变化
   - 保持健康工作
```

## 🎤 演示话术

### 开场白
```
大家好！今天我要演示 Moved Yet 插件的两个新功能：
每日健康报告和深色模式适配。

这两个功能将帮助你更好地了解和改进自己的健康习惯，
同时在舒适的深色模式下保护你的眼睛。
```

### 每日健康报告介绍
```
首先是每日健康报告功能。

这个功能会在每天首次启动 VS Code 时自动显示，
为你提供前一天的健康数据总结。

报告包含四个部分：
1. 健康评分 - 0到100分的综合评分
2. 评级系统 - 优秀、良好、一般、需改进
3. 统计数据 - 起身次数、喝水次数、工作时长
4. 健康建议 - 根据你的数据生成的个性化建议

评分算法是基于科学的健康研究设计的：
- 起身次数占40分，理想是每小时起身1次
- 喝水次数占40分，理想是每小时喝水1-2次
- 工作时长占20分，理想是4-8小时

通过这个报告，你可以清楚地了解自己的健康状况，
并根据建议持续改进。
```

### 深色模式介绍
```
接下来是深色模式适配功能。

我们对所有的 UI 界面进行了完整的深色模式适配，
包括健康仪表盘、提醒窗口、每日健康报告等。

最棒的是，这个功能完全自动，无需任何配置。
它会自动跟随 VS Code 的主题设置。

让我演示一下效果...

（切换主题）

你可以看到，所有的界面都完美适配了深色主题。
背景色、文字颜色、按钮样式都自动调整了。

这不仅让界面更美观，更重要的是保护你的眼睛，
特别是在夜间工作时。
```

### 结束语
```
以上就是 Moved Yet v0.0.2 的两个新功能。

通过每日健康报告，你可以科学地了解和改进健康习惯。
通过深色模式适配，你可以在舒适的环境下工作。

记住：先保持健康，再高效工作！

谢谢大家！
```
