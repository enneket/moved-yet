# 快速参考：新功能

## 🎯 专注模式

### 快速启用
```
命令面板 → "Moved Yet: Toggle Focus Mode"
或点击状态栏 → "🎯 进入专注模式"
```

### 功能特性
- ⏸️ 临时禁用所有健康提醒
- ⏱️ 自定义时长（1-480分钟）
- 📊 状态栏实时显示剩余时间
- 🔄 时间结束自动恢复提醒

### 配置选项
```json
{
    "movedYet.focusModeDefaultDuration": 60  // 默认时长（分钟）
}
```

### 使用场景
- 📝 重要会议期间
- 💻 深度编码时段
- 🎯 处理紧急任务
- 📚 专注学习时间

---

## 📊 每日健康报告

### 快速启用
```json
{
    "movedYet.enableDailyReport": true,
    "movedYet.enableHistory": true
}
```

### 查看报告
```
命令面板 → "Moved Yet: View Daily Health Report"
或
命令面板 → "动了么？: 查看每日健康报告"
```

### 评分标准
| 分数 | 评级 | 说明 |
|------|------|------|
| 90-100 | 🌟 优秀 | 健康习惯非常好 |
| 75-89 | 😊 良好 | 健康习惯良好 |
| 60-74 | 😐 一般 | 有改进空间 |
| 0-59 | 😟 需改进 | 需要更多关注 |

### 评分构成
- **起身次数**: 40分（理想：每小时1次）
- **喝水次数**: 40分（理想：每小时1-2次）
- **工作时长**: 20分（理想：4-8小时）

## 🌙 深色模式

### 自动适配
✅ 无需配置，自动跟随 VS Code 主题

### 适配界面
- ✅ 健康仪表盘
- ✅ 提醒窗口
- ✅ 每日健康报告
- ✅ 所有统计界面

### 切换主题
```
VS Code 设置 → 主题 → 选择深色/浅色主题
```

## 🎯 常用命令

| 命令 | 功能 |
|------|------|
| `Moved Yet: Toggle Focus Mode` | 切换专注模式 |
| `Moved Yet: View Daily Health Report` | 查看每日健康报告 |
| `Moved Yet: View Health Dashboard` | 查看健康仪表盘 |
| `Moved Yet: View Reminder History` | 查看提醒历史 |
| `Moved Yet: Show Reminder Status` | 显示提醒状态 |
| `Moved Yet: Reset All Timers` | 重置所有计时器 |

## ⚙️ 推荐配置

### 完整配置
```json
{
    // 基础提醒
    "movedYet.sitReminderInterval": 60,
    "movedYet.drinkReminderInterval": 45,
    "movedYet.enableSitReminder": true,
    "movedYet.enableDrinkReminder": true,
    
    // 渐进式提醒
    "movedYet.enableProgressiveReminder": true,
    
    // 活动检测
    "movedYet.enableActivityDetection": true,
    "movedYet.inactivityResetTime": 5,
    
    // 历史记录
    "movedYet.enableHistory": true,
    
    // 每日报告（新功能）
    "movedYet.enableDailyReport": true,
    
    // 专注模式（新功能）
    "movedYet.focusModeDefaultDuration": 60,
    
    // 语言
    "movedYet.language": "zh-CN"
}
```

### 最小配置
```json
{
    "movedYet.enableDailyReport": true
}
```

## 🐛 故障排除

### 报告不显示
1. 检查 `enableDailyReport` 是否为 `true`
2. 确认有历史数据（至少完成一次提醒）
3. 尝试手动查看命令
4. 查看控制台错误日志

### 深色模式不生效
1. 确认已编译最新代码
2. 刷新 WebView 界面
3. 重启 VS Code
4. 切换主题测试
